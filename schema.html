Generative AI Vocabulary Relationship Graph

This graph shows relationships between selected vocabulary terms and their usage in my research report on Generative AI applications. Each edge connects a term to a use case section. Edge labels and tooltips display both explicit and implicit usage frequencies, counted manually based on the current version of the report.

Legend

  • Blue nodes – vocabulary terms.
  • Green nodes – use cases from the research report.
  • Edge label – E:x / I:y where x is explicit usage count and y is implicit usage count.
  • Explicit usage – the term is directly mentioned in the section.
  • Implicit usage – the concept is used indirectly or by meaning, without exact wording.

Counts are intentionally approximate and manually evaluated as part of the learning exercise, focusing on understanding how GenAI vocabulary is distributed across different application scenarios.
